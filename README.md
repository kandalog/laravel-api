# Laravel API プロジェクト

## 環境構築

1. .env の準備
2. docker compose up -d

## バックエンド

PHP
Laravel

## フロントエンド

別途作成 (CROSS ORIGIN)

## データベース

MySQL

## 環境変数

docker/db/.env

```
MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=
MYSQL_USER=
MYSQL_PASSWORD=
```

src/.env

```
# dockerに合わせる
DB_CONNECTION=mysql
DB_HOST=
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
```
